Sau khi thử nhiều framework, tôi chọn **Astro** để xây chính website này. Đây là lý do.

## Điểm mạnh

- **Tốc độ**: mặc định xuất ra HTML tĩnh, gần như không JavaScript thừa
- **SEO tốt**: cấu trúc sạch, dễ tối ưu meta, sitemap, RSS
- **Content collections**: viết blog bằng Markdown cực gọn
- **Deploy dễ**: hợp hoàn hảo với Cloudflare Pages

## Điểm cần lưu ý

- Hệ sinh thái nhỏ hơn Next.js
- Cần làm quen với mô hình "islands" nếu muốn thêm tương tác

## Kết luận

Với một website cá nhân thiên về nội dung và SEO như vu.ai.vn, Astro là lựa chọn khó chê. Điểm: **9/10**.
